Det finns även en selektor som kan välja alla element på sidan, den s.k. universella selektorn.

Denna selektor ska du vara väldigt sparsam med och endast använda för grundinställningar på sidan, s.k "browser reset".

Ett bra exempel är att låta alla element tillämpa border-box istället för content-box.

* { box-sizing: border-box; }

Läs om skillnaden i box-sizing här

Prova nu universalselektorn för att välja alla element!
